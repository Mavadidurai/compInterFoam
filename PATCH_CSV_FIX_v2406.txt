KEY CHANGES FOR OpenFOAM v2406:
================================================================================

OLD (deprecated since 2018):
    dimensionedScalar Tvap_dim(phaseChangeDict.lookup("Tvap"));
    input_Tvap = Tvap_dim.value();

NEW (modern API - no warnings):
    input_Tvap = phaseChangeDict.get<dimensionedScalar>("Tvap").value();

This is a one-liner that:
1. Uses get<> template method (modern OpenFOAM API)
2. Returns the dimensionedScalar directly
3. Calls .value() to extract the numeric value
4. No deprecation warnings!

================================================================================

    auto Tvap_dim = phaseChangeDict.get<dimensionedScalar>("Tvap");

Or with explicit type:
    const dimensionedScalar Tvap_dim = phaseChangeDict.get<dimensionedScalar>("Tvap");
    input_Tvap = Tvap_dim.value();

All three approaches work and avoid deprecation warnings.
