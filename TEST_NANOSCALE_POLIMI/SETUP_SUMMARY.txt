QUICK START
================================================================================
1. Transfer to PoliMi HPC:
   scp -r TEST_NANOSCALE_POLIMI user@polimi-hpc.it:~/

2. Edit SLURM script:
   nano run_nanoscale_lift.slurm
   # Update: --account=your_account_name

3. Generate & decompose:
   blockMesh
   cp -r 0.orig 0
   decomposePar -copyZero

4. Submit job:
   sbatch run_nanoscale_lift.slurm

5. Monitor:
   tail -f slurm_*.out
   tail -f log.compInterFoam

6. Post-process:
   ./Allpost
   paraFoam
