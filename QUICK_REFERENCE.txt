7. KEY ALGORITHMS & FORMULATIONS
================================================================================
  PIMPLE Pressure-Velocity Coupling (pEqn.H):
  ├─ Assembly: GAMG solver on p_rgh
  ├─ Boundary pressure: p_rgh reference cell 0, value 101325 Pa
  └─ Iterations: nOuterCorrectors (10) × nCorrectors (3)

  VOF Interface Tracking (alphaEqn.H):
  ├─ Method: MULES + Crank-Nicolson or Euler ddt scheme
  ├─ Compression: interfaceCompression scheme on phir
  ├─ Sub-cycling: nAlphaSubCycles (5) for stability on tiny time-steps
  └─ Source: Phase-change mass transfer Sp, Su from twoPhaseMixtureThermo

  Phase-Change Evaporation (twoPhaseMixtureThermo.C):
  ├─ Model: Hertz-Knudsen (Clausius-Clapeyron or kinetic theory)
  ├─ Condition: Activated when alphaMin < alpha < alphaMax and T > Tvap
  ├─ Formula: Γ_evap ~ √(p_sat/(2πMkT)) × accommodation_coeff
  └─ Limit: maxPhaseChangeSource [W/m³] cap, relaxationTime temporal smoothing

  Two-Temperature Electron-Lattice Coupling (twoTemperatureModel.C):
  ├─ Electrons: Ce·∂Te/∂t + ∇·(ke∇Te) = G(Tl-Te) + Q_laser + h_gm(T_gas-Te)
  ├─ Lattice:  Cl·∂Tl/∂t + ∇·(kl∇Tl) = G(Te-Tl) - Γ_evap·hf + h_gm(T_gas-Tl)
  ├─ Sub-cycle: electronSubCycles per main time-step for stiff coupling
  └─ Bounds: minTe, maxTe, minTl, maxTl applied with temperature blending

  Recoil Pressure (advancedInterfaceCapturing.C):
  ├─ Source: Kinetic theory from evaporation rate + momentum accommodation
  ├─ Activation: Gradual ramp from Tmelt to Tvap with temperature offset
  ├─ Clamping: recoilMax pressure ceiling (typically 50–100 MPa)
  └─ Coupling: Added as RHS source in pEqn.H, gradient in UEqn.H

  Femtosecond Laser Deposition (femtosecondLaserModel.C):
  ├─ Spatial: Gaussian profile within spotSize, absorption exp(-z/δ)
  ├─ Temporal: Gaussian pulse envelope with pulseWidth FWHM
  ├─ Energy: Conserved within tolerances (pulseEnergyToleranceRel, Abs)
  └─ Fresnel: Reflectivity → transmission at interfaces automatically

8. TIME-STEPPING HIERARCHY
================================================================================
  Main Outer Loop: compInterFoam.C
    ↓ Time advance: t → t + dt (adaptive via Courant limits)
    │
    PIMPLE Iterations (nOuterCorrectors times):
      ├─ Momentum Prediction: UEqn.H (solve U implicitly)
      │
      ├─ Pressure-Velocity Correction (nCorrectors times):
      │   ├─ Non-orthogonal correction (nNonOrthogonalCorrectors times):
      │   │   └─ pEqn.H (solve p_rgh, correct U)
      │
      ├─ Interface Transport (with sub-cycling):
      │   └─ alphaEqn.H (solve alpha.metal, alpha.air, nAlphaSubCycles times)
      │
      ├─ Update Thermodynamic Properties:
      │   └─ mixture.correct() → density, etc.
      │
      └─ Energy Equations:
          ├─ TEqn.H (solve gas temperature T if exists)
          └─ twoTemperatureModel.correct() → solve Te & Tl (with e-subCycles)

    Post-Processing (if writeTime):
      └─ postProcess.H → LIFT diagnostics, recoil pressure field output
